  • WISP

    WISP это программное решение для создания корпоративного мобильного приложения для сотрудников. С помощью WISP любая компания может быстро создать канал эффективной коммуникации со своими сотрудниками.

    Решение полезно для HR менеджмента: для быстрой адаптации новых сотрудников, развития корпоративной культуры и углубления понимания процессов компании.

    Функции приложений WISP:

    • Быстрый доступ сотрудников к политике компании.
    • Всегда под рукой список контактов всех сотрудников, включающий номера телефонов, адреса эл.почты, имена в Skype, а также принадлежность к различным офисам.
    • С помощью раздела новостей можно быть всегда в курсе событий в компании.
    • Ведение рабочей коммуникации через встроенный корпоративный мессенджер.

    При создании веб-интерфейса WISP использовались следующие технологии:

    • Front-end: AngularJS, сочетание Bower, Grunt и Yeoman.
    • Back-end: ASP.NET WebApi, OWIN, Katana, MongoDB.

    Приложения WISP доступны для платформ iOS и Android. Для установки требуются версии мобильных операционных систем iOS 7 или выше и Android 4.0.3 или выше.

    Мобильные приложения созданы с использованием таких технологий как:

    • Android: Jackson, Glide, Dagger2.
    • iOS: AFNetworking, AutoScrollLabel, SDWebImage.

    Чтобы работать с WISP, необходимо зарегистрироваться на сайте сервиса, заполнить профиль информацией и выслать сотрудникам приглашения загрузить приложение для iOS или Android. Приложения скачиваются бесплатно в iTunes или Google Play.
